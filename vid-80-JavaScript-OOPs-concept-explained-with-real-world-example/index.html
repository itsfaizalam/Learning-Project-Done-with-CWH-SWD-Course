OOP in JavaScript (Hinglish Explanation)

OOP (Object-Oriented Programming) JavaScript mein ek programming style hai jo objects ko use karta hai. Object basically ek entity hoti hai jisme properties (data) aur methods (functions) hote hain. Object-oriented programming ka main focus reusability aur modularity hota hai.

OOP ke key concepts:

  • Class: Ek blueprint ya template hota hai jisse hum objects bana sakte hain.
  • Object: Class ka ek instance hota hai.
  • Inheritance: Ek class doosri class ki properties aur methods ko inherit (yaani use) kar sakti hai.
  • Encapsulation: Data aur methods ko object ke andar bundle karna.
  • Polymorphism: Same method ko different objects ke liye alag-alag tareeke se behave karne dena.
  • Abstraction: Sirf important details show karna aur baaki cheezein hide karna.

Example:

Chalo ek example ke through samajhte hain:


// Class define kar rahe hain
class Car {
  constructor(brand, model) {
    this.brand = brand;  // yeh property hai
    this.model = model;  // yeh bhi property hai
  }

  // Method define kar rahe hain
  start() {
    console.log(`${this.brand} ${this.model} ki gaadi start ho gayi!`);
  }
}

// Class ka object banaate hain
const myCar = new Car('Tesla', 'Model X');
myCar.start(); // Output: Tesla Model X ki gaadi start ho gayi!

// Inheritance ka example
class ElectricCar extends Car {
  constructor(brand, model, batteryLife) {
    super(brand, model); // super keyword se parent class ke constructor ko call kar rahe hain
    this.batteryLife = batteryLife; // ElectricCar ki apni property
  }

  // Method override kar rahe hain (Polymorphism)
  start() {
    console.log(`${this.brand} ${this.model} ki electric gaadi start ho gayi! Battery life: ${this.batteryLife}`);
  }
}

const myElectricCar = new ElectricCar('Tesla', 'Model S', '85%');
myElectricCar.start(); // Output: Tesla Model S ki electric gaadi start ho gayi! Battery life: 85%

Explanation:

- Humne Car class banaya jisme brand aur model properties hain.

- Ek start() method banaya jo console mein output deta hai.

- myCar object banaya using Car class, aur uska start() method call kiya.

- Phir inheritance ka use karke ElectricCar class banayi jo Car se inherit karti hai, aur method overriding dikhaya.

Yeh OOP ka basic idea hai JavaScript mein, jo reusability aur maintainability ko improve karta hai.

JavaScript JSON Methods Notes

What is JSON.stringify()?

JSON.stringify() is a method in JavaScript that converts a JavaScript object or value into a JSON string. It’s often used when you need to send data over a network or store it in a string format.

Example:

const person = {
    name: "John",
    age: 30,
    city: "New York"
};

const jsonString = JSON.stringify(person);

console.log(jsonString); // Output: {"name":"John","age":30,"city":"New York"}

What is JSON.parse()?

JSON.parse() is a method in JavaScript that parses a JSON string and converts it back into a JavaScript object. It’s often used when you receive data from an API or local storage in JSON format.

Example:

const jsonString = '{"name":"John", "age":30, "city":"New York"}';

const person = JSON.parse(jsonString);

console.log(person.name);  // Output: John

Notes: We can store only string in the storage, not the Object, but if we require it as an object, we can parse with JSON.parse.
